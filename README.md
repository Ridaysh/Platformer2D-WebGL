# Platformer2D-WebGL

Этот репозиторий содержит автоматически публикуемый WebGL-билд Unity-проекта.

## Играть онлайн

Игра доступна по ссылке:  
https://Ridaysh.github.io/Platformer2D-WebGL

## Описание

- Исходный Unity-проект находится в приватном репозитории Platformer2D-Private.
- При каждом push в ветку `build`:
  1. GitHub Actions запускает автоматическую сборку WebGL.
  2. Готовый билд публикуется в корень этого репозитория.
  3. GitHub Pages автоматически обновляет хостинг.

Таким образом реализован CI/CD пайплайн для Unity WebGL.

## Структура

- `index.html` - точка входа WebGL-билда  
- `Build/` - скомпилированные файлы Unity  
- `TemplateData/` - данные WebGL-шаблона  

## Назначение

Проект создан в рамках учебного задания для демонстрации:
- автоматической сборки Unity WebGL
- публикации билда через CI/CD
- хостинга через GitHub Pages
